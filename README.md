# Scripted text
Create one hotkey with scripted text + sound  
Tested in python3.6 OBS Studio version > 25.0.0 
# Usage
Create text source.
_Optionally create media source_  
- Open `Tools>Scripts`
- select this script 
- set settings for it, change duration and refresh rate
- preview it if need,
- set hotkey in `File>Settings`
